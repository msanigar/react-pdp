# react
simple react spa

# Set up

``` bash

npm i -g webpack
npm i

```

# Development
``` bash

npm run start
open http://localhost:9001

```

# Tests
``` bash

npm run test

```

# Build
``` bash

npm run build
output: /dist/

```

# Layout updte on sku

```

Magento -> Catalogue -> Manage Products -> {sku} -> Design -> Layout update

<reference name="product.info">
    <action method="setTemplate"><template>catalog/product/react-view.phtml</template></action>
</reference>

```
